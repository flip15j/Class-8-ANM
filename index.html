3. Write a Java Program to swap two numbers using third variable.
Sol.: public class SwapNumbers {
public static void main(String[] args) {
int first = 23, second = 25;
System.out.println("--Before swap--");
System.out.println("First number = " + first);
System.out.println("Second number = " + second);
int temporary = first;
first = second;
second = temporary;
System.out.println("--After swap--");
System.out.println("First number = " + first);
System.out.println("Second number = " + second);
}
}

14. Write a Java Program to check whether the number is Prime or not.
Sol.: public class Main {
public static void main(String[] args) {
int num = 29;
boolean flag = false;
for (int i = 2; i <= num / 2; ++i) {
if (num % i == 0) {
flag = true;
break;
}
}
if (!flag)
System.out.println(num + " is a prime number.");
else
System.out.println(num + " is not a prime number.");
}
}

15. Write a Java Program to calculate the Power of Number.
Sol.: class Main {
public static void main(String[] args) {
int base = 2, exponent = 4;
long result = 1;
while (exponent != 0) {
result *= base;
--exponent;
}
System.out.println("Answer = " + result);
}
